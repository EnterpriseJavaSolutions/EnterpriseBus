# EnterpriseBus
A simple, lightweight event bus for Java 1.8 (or above) applications

## How to use?
![badge](https://jitpack.io/v/EnterpriseJavaSolutions/EnterpriseBus.svg)
<br/>
Install it via JitPack, it has detailed instructions on [the website](
https://jitpack.io/#EnterpriseJavaSolutions/EnterpriseBus/).

### Basic usage
Check in the src/test/java folder for examples of how to use the bus.
